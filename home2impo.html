APA ITU IMPORTIR.ORG?

Pertanyaan POPULER Yang Sering Ditanyakan


Bagaimana cara mencari HS Code ? :

1. Bisa tanyakan suppliernya dulu, lalu pakai 4 digit pertama saja , cari di web intr.insw.go.id
2. Googling (paling gampang, mis keywordnya : Hs Code for Smartwatch), pakai 4 digit pertama juga lalu ke intr.insw.go.id
3. Untuk make sure bisa tanyakan di webchat kami selama jam kerja 9-18 senin – jumat
LAST : Pastikan HS CODE Indonesianya mengikuti standar : 8 digit

Berapa minimal pengiriman via udara dan laut?

Laut minimum 0,2 CBM, dan udara min 1 kg. Tidak diberlakukan pembulatan, system akan memakai data real misalnya 2,9657
CBM. yang penting mengikuti min. nya saja

Bagaimana proses pembayaran produk ke supplier?

Semua pembayaran harus lewat IMPORTIR.ORG, kami akan
membantu member mentransfer RMB nya ke supplier dengan
rek bank kami yang ada di cina. Kurs kami bisa diupdate di link
importir.com/exchange. Leadtimenya 3 hari kerja sudah diterima
dana nya oleh supplier

Apakah Pengiriman dari gudang IMPORTIR.ORG di Jakarta ke
alamat member dikenakan biaya ongkir ?:

Untuk Jabodetabek jika input orderan di importir.org/shipping
free ongkir, untuk diluar daerah itu ongkir ditanggung oleh
masing2 member . untuk input orderan dari importir.com
tidak free ongkir

Bagaimana cara menghitung pembelian ,HPP dan yang lain lain ?:

Di sistem kami sudah disediakan SIMULATOR SHIPPING, jadi
anda bisa tinggal masukkan datanya, maka HPP sampai Jakarta
bisa langsung anda lihat
